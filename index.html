Hướng Dẫn Lấy Chat ID Telegram

Thực hiện 1 lần duy nhất để bảo mật tài khoản

  1. Mở ứng dụng Telegram, tìm kiếm bot @userinfobot và bấm Start.
  2. Bot sẽ trả về một dãy số dòng Id (Ví dụ: 1014440614). Đây chính là Chat ID của bạn.
  3. Gửi mã Chat ID này cho Admin (Quản trị viên) để gán vào tài khoản của bạn trên hệ thống.
  4. Tiếp theo, tìm tên Bot OTP của GHN do Admin cung cấp và bấm Start để cho phép Bot gửi tin nhắn.
  5. Sau khi Admin xác nhận hoàn tất, bạn có thể đăng nhập bình thường.
